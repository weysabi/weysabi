Weysabi is local-first TypeScript infrastructure for production AI applications. It gives you one execution API across providers while keeping requests, credentials, and application data in your infrastructure.

## What Weysabi handles

- Provider dispatch, retries, circuit breaking, and fallback
- Streaming and framework adapters
- Structured output with Zod
- Typed prompt templates
- Guardrails and telemetry hooks
- Local RAG and persistent conversation memory
- An optional OpenAI-compatible server

## What stays yours

Your application owns authentication, provider credentials, databases, user authorization, retention policy, and deployment. Weysabi does not require traffic to pass through a hosted gateway.
